## Что нужно настроить

### 1. Email отправка (Production)

Установите Resend:
```bash
npm install resend
```

В `.env` добавьте:
```env
RESEND_API_KEY=your_resend_api_key
```

Раскомментируйте код в `src/lib/email/send-verification.ts`:
```typescript
// Замените console.log на:
const { Resend } = await import('resend');
const resend = new Resend(process.env.RESEND_API_KEY);

await resend.emails.send({
  from: 'AI Image Platform <noreply@yourdomain.com>',
  to: email,
  subject: 'Подтвердите ваш email',
  html: `...`,
});
```

### 2. Миграция БД (опционально)

Добавлена модель `PasswordResetToken` в `prisma/schema.prisma`.

Применить миграцию вручную:
```bash
npx prisma migrate dev --name add_password_reset_tokens
```

Или использовать временное решение (уже реализовано) через `VerificationToken`.

### 3. Дополнительные зависимости

Установлены:
- `date-fns` - для форматирования дат

## Тестирование

### 1. Профиль
```
1. Войдите в аккаунт
2. Перейдите на /profile
3. Проверьте статистику
4. Отредактируйте имя/email
5. Измените пароль
6. Проверьте историю активности
```

### 2. Email верификация
```
1. Зарегистрируйте нового пользователя
2. Проверьте консоль - там будет ссылка верификации
3. Перейдите по ссылке
4. Email должен быть подтверждён
```

### 3. Сброс пароля
```
1. На странице /login нажмите "Забыли пароль?"
2. Введите email
3. Проверьте консоль - там будет ссылка сброса
4. Перейдите по ссылке
5. Установите новый пароль
6. Войдите с новым паролем
```

### 4. История редактирований
```
1. Отредактируйте несколько изображений в /edit
2. Перейдите на /edits
3. Проверьте список редактирований
4. Откройте одно в редакторе
5. Удалите редактирование
```
