### 6. ✅ Batch генерация

**Функциональность:**

#### 🔢 Множественная генерация
- До 10 промптов за один запрос
- Параллельная обработка всех промптов
- Независимая генерация каждого изображения
- Обработка ошибок для отдельных промптов

#### 📊 Прогресс и результаты
- Прогресс-бар с процентами
- Счётчик "N/M" (текущая/всего)
- Детальные результаты:
  - ✅ Успешные генерации
  - ❌ Неудачные с описанием ошибок
- Статистика: "Успешно: X/Y"

#### 🎛️ Интерфейс
- Добавление/удаление промптов
- Отдельные текстовые поля для каждого промпта
- Нумерация промптов
- Счётчик промптов (X/10)
- Общие настройки:
  - Модель (SD/DALL-E)
  - Размер изображения
  - Негативный промпт (для SD)
- Проверка лимитов генераций перед запуском

**Файлы:**
- `src/app/api/generate/batch/route.ts` - API endpoint
- `src/components/generator/batch-generation-form.tsx` - форма с прогресс-баром
- `src/app/generate/batch/page.tsx` - страница batch-генерации
- Обновлён `src/app/generate/page.tsx` - кнопка перехода к batch

## Тестирование

### 1. Редактор - Обрезка
```
1. Откройте /edit
2. Загрузите изображение
3. Нажмите "Обрезать"
4. Измените размер выделения
5. Нажмите "Применить обрезку"
6. Изображение должно быть обрезано
```

### 2. Редактор - Кисть
```
1. Откройте /edit
2. Нажмите "Кисть"
3. Выберите цвет
4. Измените размер кисти
5. Рисуйте на canvas
6. Отключите режим кисти
```

### 3. Редактор - Слои
```
1. Добавьте несколько объектов (текст, фигуры)
2. Откройте панель "Слои" (внизу слева)
3. Попробуйте:
   - Переместить слои (стрелки вверх/вниз)
   - Скрыть/показать слой (иконка глаза)
   - Удалить слой
   - Выбрать слой кликом
```

### 4. Batch-генерация
```
1. Перейдите на /generate
2. Нажмите "Batch генерация"
3. Добавьте 3-5 промптов
4. Нажмите "Сгенерировать все"
5. Наблюдайте прогресс-бар
6. Проверьте результаты
7. Перейдите в историю генераций
```
